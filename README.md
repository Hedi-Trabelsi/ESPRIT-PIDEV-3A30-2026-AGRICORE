# Agricore – Desktop Application for Agriculture Management

## Overview
This project was developed as part of the PIDEV – 3rd Year Engineering Program at **Esprit School of Engineering** (Academic Year 2025–2026).  
Agricore is a desktop application designed to help farmers manage various aspects of their agricultural business efficiently. The application includes six main modules to cover users, animals, equipment, events, finance, and maintenance.

## Features
- **Users**: Manage farmers and staff accounts.  
- **Animals**: Track livestock and health status.  
- **Equipment**: Manage tractors, tools, and other farm machinery.  
- **Events**: Organize and monitor agricultural events .  
- **Finance**: Track expenses, revenues, and financial reports.  
- **Maintenance**: Schedule and record maintenance for equipment.

## Tech Stack
### Frontend
- JavaFX (Desktop GUI)
### Backend
- Java
- XAMPP / phpMyAdmin (for database management)

## Architecture
- Modular design with 6 main modules interacting via a central database.  
- MVC pattern implemented for separation of concerns.
